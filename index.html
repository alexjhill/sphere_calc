What's this all about?

As a broad definition, Monte Carlo simulations are computational algorithms which use randomness to find the answers to problems which are considered to be deterministic. Their practical uses range from x to y. This visualisation demonstrates how a Monte Carlo simulation can be used to approximate the area of a circle without the use of π, as well as approximate the value of π to a surprising degree of accuracy.
